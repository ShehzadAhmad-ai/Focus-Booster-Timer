Focus Booster Timer - Productivity Tool

Overview
A clean, minimalist focus timer designed to help you maintain concentration and manage work/break intervals effectively. This web-based tool allows for customizable focus sessions with visual progress tracking and essential productivity features.

Features
Customizable Timers: Set work and break durations to match your workflow

Visual Progress Bar: Track session completion with a dynamic progress indicator

Dark/Light Mode: Toggle between themes for comfortable use in any lighting

Session Tracking: Count completed work sessions with persistent local storage

Motivational Quotes: Random inspirational quotes display during break times

Responsive Design: Fully responsive layout for desktop, tablet, and mobile

Audio Notifications: Gentle sound alerts when sessions change

Simple Controls: Easy-to-use start, pause, and reset functions

Technologies Used
HTML5

CSS3 (Flexbox, CSS Variables)

Vanilla JavaScript

Local Storage API

Responsive Web Design

Web Audio API

How to Use
Set Durations: Adjust work and break times using the number inputs

Start Timer: Click "Start" to begin your focus session

Track Progress: Watch the visual progress bar fill as time elapses

Take Breaks: The timer automatically switches to break mode when work ends

Toggle Theme: Switch between light and dark modes for comfortable viewing

View Stats: Track your completed sessions (persists across browser sessions)
